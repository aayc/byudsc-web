BYU DSC Website Admin Panel

What problem does this solve?

Editing HTML pages and knowing where to put them, and especially how to link them together, is a pain! Write your HTML here (or write it in a text editor - vim, Sublimetext, whatever, and paste it here) and let the admin panel do the linking work for you.

What's in here?

There are two main modules - the Technique and Event managers (write up manager coming soon!). The Technique manager lets you a) create articles and post them under categories, b) create and delete categories, and delete articles. The Event manager lets you a) create events and post them under date categories, b) create and delete categories, and delete events.

How does the Admin panel work, technically speaking?

Different panels work differently. The Events panel pushes the events you create into events.json. Nodejs pulls from events.json to create the static HTML page you see in the Events tab of the main website.

What if I lose my data?

I trust you. Make backups. I provided a backup utilities for you. One of them (backup-the-json.py) will zip the current state of the json files and place it into the json-backup folder. Use this liberally.
