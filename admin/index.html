BYU DSC Website Admin Panel

What problem does this solve?

Editing HTML pages and knowing where to put them, and especially how to link them together, is a pain! Write your HTML here (or write it in a text editor - vim, Sublimetext, whatever, and paste it here) and let the admin panel do the linking work for you.

What's in here?

There are two main modules - the Technique and Event managers (write up manager coming soon!). The Technique manager lets you a) create articles and post them under categories, b) create and delete categories, and delete articles. The Event manager lets you a) create events and post them under date categories, b) create and delete categories, and delete events.

How does the Admin panel work, technically speaking?

It's kind of a hack, to be honest. Essentially, for each Manager, there's a JSON file that holds the structure. For example, for the Events manager, the JSON file will have an array of category objects, each holding a series of events, etc. The JSON file and the website are not directly connected! When you click "update website", an HTML page will be created from the JSON file (from a template), and that new page will replace the current version on the site. Your written article will also appear in the public/ folder under one of the storage folders, (technique-articles/, for example).

This is why there are both the "Save" and the "Update website" buttons. Saving saves the JSON file, updating the website applies it.

What if I lose my data?

You'll want to keep back ups - this is a "I'm trusting you" system. I provided a backup utility (backup-the-website.py) in the root folder. Click on it and it will zip the public folder in its current state and place it under the "public-backups" folder. As for the JSON files, there's a "backup-the-json.py" file in this directory - this will zip all the necessary json files and put them under the "json-templates-backups" folder. Your articles will be placed in the "/admin/content/" folder, and you can only delete them explicitly here. You can unlink them without deleting them too.
